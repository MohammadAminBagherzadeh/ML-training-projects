These training projects form the foundation of my machine learning portfolio and demonstrate my readiness to pursue more advanced studies or professional roles in machine learning and artificial intelligence.

The goals of this repository are to:

# Project Structure

The repository is organized into individual project folders or notebooks. Each folder contains a focused machine learning task, including:

**Regression**

Simple Linear Regression
Multiple Linear Regression
Polynomial Regression
Non-Linear Regression
Tehran House Prices (Regression example using real data gathered from Tehran city in Iran)

These projects cover numerical prediction tasks and help build intuition for continuous target modeling.

**Classification**

Logistic Regression
SVM Classification
k-Nearest Neighbors (kNN)
Decision Trees Classification
Risk of Heart Attack (Binary Classification example using real life data)

Classification projects show understanding of binary and multiclass decision boundaries.

**Clustering**

K-Means Clustering
Hierarchical Clustering
DBSCAN Clustering
Customer Clustering (Using real life data found on Kaggle)

These demonstrate unsupervised learning for grouping similar data.

**Recommendation Systems**

User-Similarity Recommendation (Trained on Netflix's data)
Content-Based Recommendation (Trained on Netflix's data)
Phone Recommender System (Using real life data found on Kaggle)

These projects explore approaches for suggesting items based on data patterns.

Navigate into a project folder
